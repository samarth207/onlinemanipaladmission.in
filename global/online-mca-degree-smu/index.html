Semester 1

Computational Mathematics

Learners will understand fundamental computational concepts and develop the ability to implement and analyze algorithms. They will also learn to design efficient algorithms and evaluate the most suitable ones for solving specific problems.

Java Programming

Learners will grasp the core concepts of Object-Oriented Programming (OOP) and apply them to develop complex software solutions. They will also learn to use Java libraries, design user interfaces, and evaluate the efficiency of their Java code.

Operating Systems

Learners will understand fundamental algorithmic problem-solving techniques and analyze the efficiency of different algorithms. They will also develop, implement, and evaluate algorithms to effectively solve real-world computing problems.

Database Management System

Learners will understand core principles of databases and database management systems and apply SQL for data handling and integrity. They will also learn to optimize queries, manage concurrency, and design efficient, robust database models and applications.

Database Management System Lab

Learners will gain hands-on experience with core concepts of Database Management Systems and learn to apply SQL to perform various database operations effectively.

Java Programming Lab

Learners will gain hands-on experience with Object-Oriented Programming (OOP) concepts and learn to code in Java for practical applications, such as creating programs to manage and display employee details.

Fundamentals of Computers & IT

Learners will understand fundamental concepts of computers and IT, including number systems and logic circuits. They will also learn basic web design, CSS, web publishing, and gain hands-on experience in writing, compiling, and debugging C programs.

Fundamental of Mathematics

Learners will understand fundamental mathematical concepts and apply them in computer applications. They will also learn calculus, vectors, matrices, and complex numbers, gaining skills to perform various mathematical operations and representations.
Semester 2

Python Programming

Learners will gain hands-on experience coding in Python and manipulating data efficiently. They will also learn to handle errors, perform web scraping, and visualize data effectively using Python libraries.

Software Engineering and Unified Modelling Language

Learners will understand the fundamental principles of software engineering and apply them in software design, project management, and development. They will also analyze methodologies, evaluate project strategies, and create effective software systems.

Computer Organization and Architecture

Learners will understand computer architecture and operational concepts, including memory organization and processing techniques. They will also learn about high-speed and secondary memory, peripherals, multiprocessing, and parallel system organization.

Data Structure and Algorithm

Learners will grasp the core concepts of data structures and algorithms and implement structures such as linked lists, stacks, queues, trees, and graphs. They will also assess algorithm efficiency, design optimized solutions, and understand principles of advanced algorithmic topics.

Data Structure and Algorithm Lab

Learners will design and analyze the time and space efficiency of data structures and identify the most suitable structure for a given problem. They will also gain hands-on experience implementing data structures such as arrays and singly linked lists.

Python Programming Lab

Learners will gain hands-on experience applying Python to topics from theory subjects. They will also learn to code for data wrangling, aggregation, web scraping, and visualization using Python.
Semester 3

.NET Framework

Learners will understand and apply the architecture and components of the .NET framework, using OOP concepts in VB.NET and C#. They will also learn data access techniques, state management for web applications, and develop multi-threaded applications with the Task Parallel Library.

Computer Network

Learners will understand the fundamental concepts of OSI and TCP/IP models and implement various network protocols, addressing schemes, and routing algorithms. They will also analyze network traffic, troubleshoot connectivity issues, and compare the performance of different network designs and protocols.

IT Laws and Practices

Learners will understand the design constraints, requirements, and technologies behind mobile application development. They will also learn to apply UI and software architecture principles, differentiate between native and cross-platform approaches, implement security and performance optimization, and build fully functional mobile apps using emerging technologies.

.NET Framework Lab

This practical lab helps learners gain hands-on experience in C# programming. It also familiarizes them with the .NET framework and its applications.

Computer Network Lab

This lab activity gives hands-on experience with the network layer of OSI and TCP/IP models, teaching the use of tools and techniques for real-world networking. Learners will simulate protocols, routing algorithms, and design networks using network simulation tools to build practical skills.

Elective I (Choose any one)

Data Warehousing and Data Mining

This subject equips learners with the principles and techniques of data warehousing and data mining, enabling them to extract, analyze, and evaluate large and diverse datasets. It also covers designing efficient ETL processes and applying appropriate tools to handle structured, semi-structured, and unstructured data.

Cloud Computing

This subject introduces learners to cloud computing principles, models (IaaS, PaaS, SaaS), and virtualization, enabling effective management of cloud environments. It also covers evaluating cloud platforms, implementing security measures, and leveraging cloud technologies for business innovation and transformation.
Semester 4

Mobile Application Development

This subject equips learners with the skills to design, develop, and optimize mobile applications, covering user interface design, software architecture, and security. It also helps them understand native vs. cross-platform approaches and integrate emerging technologies to build fully functional apps.

Angular JS, React JS, and VUE JS

This subject equips learners with a solid understanding of Angular, React, and Vue JS, including their architecture, functionality, and use cases. Learners will gain hands-on skills to build, manage, and optimize dynamic web applications across these popular JavaScript frameworks.

Project

This subject enables students to apply their learning in a practical setting by undertaking a project in their chosen specialization. It aims to develop industry-ready skills through hands-on experience in handling, analyzing, and completing real-world projects.

Elective II (Choose any one)

Machine Learning

This subject introduces the fundamental concepts, principles, and algorithms of machine learning, covering supervised and unsupervised learning techniques. Learners will analyze real-world datasets to design, implement, evaluate, and synthesize effective machine learning models for diverse applications.

Distributed System and Grid Computing

This subject provides a comprehensive understanding of distributed systems, including their concepts, challenges, architectures, and emerging trends. Learners will analyze and design efficient, secure distributed solutions using communication, synchronization, concurrency control, and distributed computing principles across real-world systems.

Eligibility for online MCA

Educational qualification

Candidates must have completed 10+2+3-year bachelor’s degree in Computer Applications/Computer Science/Information Technology from a recognized university/institution, or an equivalent qualification as recognized by the Association of Indian Universities (AIU) or other competent bodies

 

Candidates who have completed 10+2+3-year bachelor’s degree from other streams like Science, Business Administration, Business Management, Arts & Humanities, Commerce, etc. are also eligible for MCA admission but they have to attend and complete a compulsory bridge course in fundamentals of computer and IT along with their Semester 1 courses. Additionally, candidates who have not studied mathematics during their 10 + 2/ bachelor’s degree programs must attend and complete a compulsory bridge course in mathematics along with other semester 1 courses.

 

Grades

Candidates must have a minimum of 50% marks in aggregate in graduation (45% for reserved categories).

Candidates who have completed their 10+2 or graduate level education outside India must produce a certificate of equivalence issued by the Association of Indian Universities.

 

Grades

Other requirements

Candidates who are not Indian citizens or residing outside India and holding NRE or PIO status will be billed an international fee for respective degrees & may need to submit documents mentioned but not limited to CV, address proof, valid visa, PR card & passport copy as part of the application process.

Are online MCA classes more affordable than regular MCA classes?

Online MCA programs are typically less expensive than on-campus MCA programs. They usually have lower tuition fees and are free from additional expenses such as relocation, commuting, and campus housing. Online MCA also provide flexible payment schedules and permit students to maintain their jobs during studies, which can help ease the financial burden.

More FAQs

Can students take online examinations from any device?

Students can take up online examinations from any desktop or laptop running a Windows Operating System with the latest version of the Google Chrome browser installed. The Safe Exam Browser (exam portal) is not compatible with iOS devices.

Is the program fee inclusive of resit and revaluation charges?

No. Additional charges may be payable for revaluation, resit examinations, duplicate certificates, and other supplementary academic services. These fees are not included in the program fee and are levied as per the respective university’s guidelines and policies.

What is the evaluation & assessment pattern? 

  • A student’s performance evaluation will be based on both continuous assessments (CA) through assignments and term-end examinations (TEE) which consist of descriptive and multiple-choice questions. A student must score at least 40% in each of these to be declared as having passed the exam. The assessment ratio of TEE to CA is 70:30.
  • A student must score at least 40% in both the parts to be declared as having passed.

What is the examination pattern for online degrees at Sikkim Manipal University? 

  • The University term-end examination (TEE) will be held digitally viz. computer-based and online proctored.
  • Candidates can appear for the exams from the comfort of their homes.
  • The details of the examination schedule will be made available in the photo admit card. Other details on the examination pattern are available in the program guide provided to students.
  • Examination results will be published on the student portal. If a student fails to clear any of the papers (either in UE or IA or both), he must re-appear by applying to the University through the re-sitting application along with relevant fees, within the due date.
  • On satisfying all criteria stipulated by the University, successful students will be awarded their respective program degree by the University.
More FAQs

How are Sikkim Manipal University’s online degree programs delivered? 

Sikkim Manipal University’s online programs are delivered in a 100% online mode so that students can attend live classes and complete the program remotely, while also pursuing their professional and personal commitments in parallel.

Will the program have live lectures and webinars? 

Yes. The program is delivered through a mix of live lectures, pre- recorded videos and discussion forums in which students can interact directly with faculty and self-paced learning. Self-paced learning is driven by video lectures, e-books, quizzes, and assignments that are delivered using a state-of-the-art Learning Management System (LMS).

Are there any mandatory sessions? 

Students are expected to attend live classes, watch recorded sessions, and go through at least 75% of the self-learning content (reading material and video lectures) available on LMS. This would ensure they are eligible to appear for the final exams.

When will I get access to the learning modules? 

After admission is confirmed by the university, students will receive access to the Learning Management System (LMS), student portal and e-library. Students can then start accessing the course material and begin their learning journey.

What are the study materials given to learners of online degree programs at Sikkim Manipal University? 

Students are given access to an exhaustive e-library, video lectures, audio-visual interactive material, virtual classroom sessions, audio podcasts, virtual simulations, and self-assessment quizzes or tests. Students are free to download e-books and print them for convenience.

How can I get my academic queries answered? 

Students will be assigned dedicated course mentors and coordinators who can be contacted to clarify doubts using discussion forums, instant messaging, and email on the Learning Management System (LMS).

How much time should I dedicate to complete the learning activities? 

Although the learning is flexible, students are expected to spend at least 15-20 hours per week watching concept videos, reading learning material, attempting quizzes, and attending live webinars. This would ensure that you get your degree within the minimum duration.

Does WES (World Education Services) evaluate online degrees from Sikkim Manipal University?

Sikkim Manipal University degrees issued after 2016 are evaluated by WES (World Education Services).

Is this online MCA program compliant with AICTE norms?

Yes, this MCA program is compliant with AICTE norms. However, as per the University Grants Commission (UGC), Central, State and Private Universities offering Undergraduate, Postgraduate & Postgraduate Diploma programs under the disciplines of Management, Computer Applications and Travel & Tourism in online mode do not require AICTE approval.
